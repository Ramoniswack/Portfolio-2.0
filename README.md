# R.a.mohan Tiwari's Portfolio

Modern portfolio website with smooth animations and optimized performance.

## Quick Start

```bash
npm install
npm run dev
```

Visit [ramohan.com.np](https://ramohan.com.np) or [localhost:3000](http://localhost:3000)

## Features

- Custom CSS cursor
- Video previews
- Mobile-responsive design
- Smooth animations

## Tech Stack

Next.js 15 • TypeScript • Tailwind CSS • GSAP
